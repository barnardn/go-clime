Uses the OpenWeatherMap API to retrive the current weather conditions.

Usage:
  go-clime [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  geocode     Fetch geographic details based on ip address
  help        Help about any command
  here        Returns the weather conditions at the current location
  weather     Returns the current weather conditions at the provide zip code

Flags:
      --config string   config file (default is $HOME/.go-clime.yaml)
  -h, --help            help for go-clime
